Neon:circle (options)
Create neon circle shape objects

Parameters:

  • options

    a table with all options for the circle

    • parent: (GroupObject) the parent object (default: nil)
    • x: (number) the center x position of the circle (default: 0)
    • y: (number) the center y position of the circle (default: 0)
    • radius: (number) the radius of the circle (default: 50)
    • strokeWidth: (number) the stroke width of the inner body (default: 2)
    • glow: (number) the size of the glow effect (default: 64)
    • glowColor: (rgba table) the color of the glow effect (default {1,1,1,1})
    • antiAlias: (bool) if antiAlias simulation for the inner body should be used (default: false)
    • showFlare: (bool) if an extra flare effects should be drawn (default: false)
    • flickerEffect: (bool) activate flickering effect (default false)
    • flickerStyle: (number) choose between flicker style 1, 2, 3, 4 or 5 (default 1) 1 is a light vibrating flickering, 2, 3 and 4 add some hard short flickers, where 2 does some seldom hard flicker and 4 does often hard flicker. 5 is most of the time dark, and sometime lightens up.

Usage:

      require("devilsquid.neonshapes.neon")
  local circle = Neon:circle({x=450,y=400, radius=100, strokeWidth=12, glow=96, antiAlias=true, showFlare=true})
  circle:setGlowColor(1,0,0,1)
  circle:setFlareRotation(-12)
  circle:setFlarePosition(-25,-97)

Neon:text (options)
Create neon bitmap text object

Parameters:

  • options

    a table with all options for the text

    • parent: (GroupObject) the parent object (default: nil)
    • x: (number) the center x position of the circle (default: 0)
    • y: (number) the center y position of the circle (default: 0)
    • text: (string) the text to display (default: ‘Lorem ipsum’)
    • font: (string) the name of the font. Right now you have two options: “NeonFontClean” and “NeonFontDistorted”
    • fontSize: (number) the size of the text (default: 80)
    • glowColor: (rgba table) color table with r,g,b,a components (default {1,0,0,1})
    • flickerEffect: (bool) activate flickering effect (default: false)
    • flickerStyle: (number) choose between flicker style 1, 2, 3, 4 or 5 (default 1) 1 is a light vibrating flickering, 2, 3 and 4 add some hard short flickers, where 2 does some seldom hard flicker and 4 does often hard flicker. 5 is most of the time dark, and sometime lightens up.

Usage:

      require("devilsquid.neonshapes.neon")
  local text = Neon:text({text="DevilSquid", x=430,y=700, fontSize=120, glowColor={1,0,0,1}, flickerEffect=true})

Neon:bezier (options)
Create and draws a cubic Bézier path with neon effects

Parameters:

  • options

    a table with options for the Bézier curve

    • parent: (GroupObject) the parent object (default nil)
    • x: (number) the x position of the path (default 0)
    • y: (number) the y position of the path (default 0)
    • controlPoints: (array) an array with a continuous list of control point coordinates like
    • glow: (number) the size of the neon glow effect (default 64)
    • glowColor: (rgba table) the color of the glow effect (default {1,1,1,1})
    • glowDensity: (number) the density of the glow effect (default 100)
    • glowIntensity: (number) the intensity (alpha) of the neon glow effect (default 0.8)
    • strokeWidth: (number) the stroke width of the inner body (default 2)
    • bodyDensity: (number) the density of the inner body (default 100)
    • bodyIntensity: (number) the intensity (alpha) of the inner body (default 0.8)
    • antiAlias: (bool) if antiAlias simulation for the inner body should be used (default false)
    • flickerEffect: (bool) activate flickering effect (default false)

Usage:

      local BezierPath = require( "devilsquid.util.bezier.bezierpath" )

  local squidBody = NeonBezierPath:new({
     bodyDensity = 200,
     glowDensity = 30,
     glow = 64,
     glowColor = {0,0,1,1},

     strokeIntensity = 0.3,
